Parts of a web page

Nearly every web page is divided into four parts:

  • header
  • footer
  • sidebar
  • content area

Divs

  • Divs let us divide web pages into relevant parts — the 4 main parts and smaller areas, as well.
  • With divs, we group elements so they can be styled the same way
  • We give divs ids and classes so we can style them using CSS.

Example: <div id="header">

Other elements are often nested inside divs

<div>
   <p>Content<p>
   <p>Content<p>
</div>
<div id="header">
   <h1>Main Heading<h1>
</div>
<div class="sub-content">
   <p>Some more content<p>
</div>

Case Study:

</div>

Padding, Borders, and Margins

Padding

Space between the border and the content

Padding

Space between the border and the content

Adds to the total width of the box.

Padding

To make the padding on all sides the same, use the "padding" property and one value


padding: 15px;

Or you can style the sides individually


padding-top: 10px;

Padding shorthand

You can also use shorhand to style the sides in one declaration


padding: 10px 5px 3px 5px;

is the same as: 


padding-top: 10px;
padding-right: 5px;
padding-bottom: 3px;
padding-left: 5px;

The order is: top right bottom left

Padding Shorthand

If the top and bottom padding are the same, and the left and right are the same, you can use two values:


padding: 30px 50px;

is the same as: 


padding-top: 30px;
padding-right: 50px;
padding-bottom: 30px;
padding-left: 50px;

The order is: top/bottom right/left

Border

The edge around the box, specified as "thickness, style, color."

Borders

  • border-width
  • border-style
  • border-color

Example: 


  border-width: 10px;
  border-style: dashed;
  border-color: #666666;

Border style shorthand

You can specify each property separately, or all three together.

A solid red border


  border: 1px solid #ff0000;

Border style shorthand

You can specify separate styles for the top, left, right or bottom border

A thick dotted black top border


  border-top: 4px dotted #000000;

Two different border styles


  border-top: 1px solid #ff0000;
  border-bottom: 4px dotted #000000;

Margin

The transparent area around the box that separates it from other elements.

Margins

Styled just like padding!


margin: 15px;

Or you can style the sides individually


margin-top: 10px;

Margin shorthand

Same as padding!


margin: 10px 5px 3px 5px;

is the same as: 


margin-top: 10px;
margin-right: 5px;
margin-bottom: 3px;
margin-left: 5px;

The order is: top right bottom left

Margin Shorthand

Like padding, if the top and bottom margin are the same, and the left and right are the same, you can use two values:


margin: 30px 50px;

is the same as: 


margin-top: 30px;
margin-right: 50px;
margin-bottom: 30px;
margin-left: 50px;

The order is: top/bottom right/left

Auto Margin

If a margin is set to auto on a box that has width, it will take up as much space as possible.

CENTERED


  margin: auto;
  width: 300px;

FLUSH-RIGHT


  margin-left: auto;
  margin-right: 5px;
  width: 300px;

Width and height

width: 600px;

Sets the width of an element to 600px.

height: 300px;

Sets the height of an element to 300px.

These only refer to the content area. Padding and border will add to the height and/or width.
